1. Probabilitas Prior

Menghitung peluang masing-masing kelas secara umum dari total {{ hasil.prior.total }} data training.

P(Membeli = Ya) = {{ hasil.prior.jml_ya }} / {{ hasil.prior.total }} = {{ "%.2f"|format(hasil.prior.ya) }}

P(Membeli = Tidak) = {{ hasil.prior.jml_tidak }} / {{ hasil.prior.total }} = {{ "%.2f"|format(hasil.prior.tidak) }}

2. Likelihood

Menghitung probabilitas setiap atribut berdasarkan kelasnya.

Kelas: Ya (n = {{ hasil.prior.jml_ya }})

  • P(Usia={{ hasil.data_uji.Usia }} | Ya) = {{ hasil.likelihood.ya.usia_count }} / {{ hasil.prior.jml_ya }} = {{ "%.3f"|format(hasil.likelihood.ya.usia) }}
  • P(Penghasilan={{ hasil.data_uji.Penghasilan }} | Ya) = {{ hasil.likelihood.ya.penghasilan_count }} / {{ hasil.prior.jml_ya }} = {{ "%.3f"|format(hasil.likelihood.ya.penghasilan) }}
  • P(Status={{ hasil.data_uji.Status }} | Ya) = {{ hasil.likelihood.ya.status_count }} / {{ hasil.prior.jml_ya }} = {{ "%.3f"|format(hasil.likelihood.ya.status) }}

Kelas: Tidak (n = {{ hasil.prior.jml_tidak }})

  • P(Usia={{ hasil.data_uji.Usia }} | Tidak) = {{ hasil.likelihood.tidak.usia_count }} / {{ hasil.prior.jml_tidak }} = {{ "%.3f"|format(hasil.likelihood.tidak.usia) }}
  • P(Penghasilan={{ hasil.data_uji.Penghasilan }} | Tidak) = {{ hasil.likelihood.tidak.penghasilan_count }} / {{ hasil.prior.jml_tidak }} = {{ "%.3f"|format(hasil.likelihood.tidak.penghasilan) }}
  • P(Status={{ hasil.data_uji.Status }} | Tidak) = {{ hasil.likelihood.tidak.status_count }} / {{ hasil.prior.jml_tidak }} = {{ "%.3f"|format(hasil.likelihood.tidak.status) }}

3. Posterior Probability

Mengalikan Prior dengan semua nilai Likelihood untuk masing-masing kelas (tanpa normalisasi pembagi).

P(Ya | X) ∝ P(Ya) × P(Usia|Ya) × P(Penghasilan|Ya) × P(Status|Ya)

= {{ "%.2f"|format(hasil.prior.ya) }} × {{ "%.3f"|format(hasil.likelihood.ya.usia) }} × {{ "%.3f"|format(hasil.likelihood.ya.penghasilan) }} × {{ "%.3f"|format(hasil.likelihood.ya.status) }}

= {{ "%.5f"|format(hasil.posterior.ya) }}

P(Tidak | X) ∝ P(Tidak) × P(Usia|Tidak) × P(Penghasilan|Tidak) × P(Status|Tidak)

= {{ "%.2f"|format(hasil.prior.tidak) }} × {{ "%.3f"|format(hasil.likelihood.tidak.usia) }} × {{ "%.3f"|format(hasil.likelihood.tidak.penghasilan) }} × {{ "%.3f"|format(hasil.likelihood.tidak.status) }}

= {{ "%.5f"|format(hasil.posterior.tidak) }}
